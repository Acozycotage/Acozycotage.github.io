1.技术栈&框架🧪

1. 框架：Hexo 6.3.0
2. 主题：Butterfly
3. 源码仓库：Github
4. 域名：腾讯云
5. SSL证书：GlobalSign旗下AlphaSSL签发的通配符SSL证书 + 萌咖杂货店token
6. 解析服务：DNSPod帝思普
7. 托管与部署：本网站采用多线部署，主部署在腾讯云CVM云服务器上，其余托管Github Page + Vercel + Netlify + Cloudflare + Railway
8. 评论系统：Twikoo系统 + 私有部署 + 腾讯云提供存储服务
9. 资源池：腾讯云COS + LightPicture + 腾讯云云服务器 + 腾讯云CDN私有部署
10. CDN：腾讯云CDN + 网盾星球 + Cloudflare
11. 图片压缩：Cloudconvert + 腾讯云COS数据万象
12. 字体压缩: Cloudconvert压缩为woff2格式 + 私有部署资源池提供线上调用
13. 静态资源压缩: Gulp + 私有部署图床提供线上调用
14. 自动部署：Git钩子 + Github Action
15. SEO优化：谷歌 + 必应 + 百度
16. 音乐平台：hexo-tag-aplayer
17. 流量监控：腾讯云云监控+51la + 灵雀监控 + 宝塔面板监控
18. 友链朋友圈：hexo-circle-of-friends爬虫 + SQLite提供存储服务数据库 + 私有部署
19. 微博热搜：weibo-top-api爬虫
20. 博客统计 ：腾讯云全球应用加速 + 百度统计 + 不蒜子
21. 天气预报：和风天气
22. 异步加载与缓存加速技术：Pjax + ServiceWorker参考空梦大佬
23. 开发语言：HTML5 + CSS3 + JavaScript + Pug + YML等
24. 性能优化：由Gulp压缩 + 私有资源池 + 腾讯云COS + 腾讯云CDN通过线上调用实现加速
25. 备份服务：宝塔面板每小时自动备份5份同步至百度网盘
26. 版权声明：详见CC BY-NC-SA 4.0
27. 维护日志：见时光回廊栏目

2.自我介绍

欢迎来到我的博客，目前18岁（周），目前是一名计算机专业的中专生，也是一名共青团员，为啥是中专生呢，中考考了，642，但我没去高中，当然了我是自愿放弃高考的，在河北这个地方，高考竞争力，说实话真的没兴趣，压力太大，高中老师1半年差不多，讲完3年的课程，然后就是无穷的刷题，刷题，刷题，每日枯燥乏味的生活，虽然也是为了将来，但是将来的生活按照这个轨迹走，一如既往的结婚生子，周而复始，这样的生活并不是我想要的，我想自己闯荡一次，目前有一家个体工商户，本人可以熟练操作的Adobe全家桶，PR算是最熟练的，中专的时候老师天天追着做这个，做那个，手里也有不少特效和素材，未来重心大概会在视频剪辑上面，做一名视频剪辑者！

3.关于博客

关于博客的话，也是刷B站意外看到猕猴桃大佬的视频，然后就突发奇想，就入坑了，哈哈 ✋~ ，当然，在此之前，我是学过前端一小部分，手里还有几本书，但是折腾下来也算是学习到了很多有用的知识怎么虽然也是也算是我记生活的地方，目前准备筹备4月份单招考试，⚡未来我会尽量进行更新 ☁️，如果没更的话，可能是我忘了，也可能是我在钻研某个东西的时候太入迷了肯定又熬夜了，不要学我，老是熬夜会长痘，现在魔改初期，几乎每晚都是从晚上折腾到天亮，一遍遍的炸，一遍遍的重试，最终发现问题原因，然后解决问题，这大概就是我目前的快了吧，在绝境中重生，一遍又一遍，关于博客内容的话：Adobe相关的内容，MAYA之类的，MC相关的知识，本人做的笔记与总结，主要是关于Hexo博客搭建和魔改、 当然也有部分东西是搬过来的(均经过作者同意并注明了出处)，还有日记，闲谈之类的
